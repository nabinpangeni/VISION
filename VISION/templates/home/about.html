VISION

The main purpose of our project is to provide a website for all those users which can provide resources as per needed to the users demand like in text, speech, summarization or in their native language. Using this websited any users around the world has full and complete access to the resources available to the websites on any of the format they are feasible and suitable.
This project uses a NLP, NLTK which helps to summarize, convert into different languages and in speech format. When a resources is uploaded to the website, system automatically synchorize it into various text, speech and summarize it. After the successful conversion of the resources, it can be accessed and utilized it according to the need of the users. Adding more on this system also rates the comment according to the sentiment analysis which is also a key factor in our project that shows the sentiment of comment.

OBJECTIVE


This "Learning with NLP" project aims to achieve the following objective:
1. To respond machine as human does
2. To processing of the text’s data
3. To convert the text’s data into audio format
4. To make sense of the human language in the manner that is valuable
5. To convert the data of one language into another language
